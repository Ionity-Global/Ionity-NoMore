# Contributing

Public source availability does not grant permission to copy, modify, or redistribute this project. Contact Ionity Global (Pty) Ltd through [ionity.co.za](https://www.ionity.co.za) before preparing a contribution.

Authorized contributions must preserve the consent and privacy model:

- no background or covert location collection;
- no analytics, advertising, tracking SDK, cloud map, or remote geocoding;
- no signaling, STUN, TURN, message, or location server without an explicit architectural and privacy review;
- no SMS inbox, call-log, contacts, or accessibility permissions;
- exact location remains explicit, visible, temporary, and higher-risk;
- carrier actions remain allowlisted, previewed, confirmed by the user, source-dated, and physically tested.

Before submitting authorized work, run `pnpm run build`, `pnpm run lint`, and `pnpm test`.
